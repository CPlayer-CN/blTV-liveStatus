# B 站主播直播检测
检测哔哩哔哩主播是否开播。
check network anchor live status of bilibili.

# 使用
1. 在终端中运行。
2. 脚本默认会每分钟刷新1次，如果在直播就返回`1`，如果不在直播就返回`0`，你还可以运用`while status == 1:`来执行你想要的效果。
